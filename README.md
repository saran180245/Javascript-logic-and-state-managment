# To-Do List Application

A simple and interactive To-Do List application built using HTML, CSS, and JavaScript. This project demonstrates JavaScript logic, state management, DOM manipulation, event handling, and local data persistence using Local Storage.

## Features

- Create new tasks
- View all tasks
- Edit existing tasks
- Delete tasks
- Mark tasks as completed
- Filter tasks by:
  - All
  - Active
  - Completed
- Persistent data storage using Local Storage
- Dynamic DOM rendering
- Event delegation for efficient event handling

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)

## Project Structure

```text
todo-app/

```

## How It Works

1. Enter a task and click **Add**.
2. Mark tasks as completed using the complete button.
3. Edit or delete tasks as needed.
4. Filter tasks using All, Active, and Completed options.
5. Tasks are automatically saved in Local Storage and remain available after page refresh.
